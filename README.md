Convert My Images
This is a tool to convert Images in Bulk. You just need to run the tool with the parameters required and everything will be done.

Supported Image Types:
PNG, JPEG, PPM, GIF, TIFF and BMP


Installation Steps:
1. Clone the Repository to your Machine using the command below:
git clone git@github.com:ajayagrawalgit/Convert-My-Images.git

2. Copy-Paste the commands below on to your Terminal:
cd Convert-My-Images;
./install.sh
